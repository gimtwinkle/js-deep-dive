# 05 표현식

### 값
+ 표현식의 결과
  
### 리터럴
+ 값이나 변수가 아닌 고정된 값
  
### 표현식
+ 값으로 평가되는 문
  
### 문
+ 프로그램의 최소단위, 더이상 나눌 수 없음
+ ex)
  ```javascript
  var sum; //var;  또는 sum; 으로 나눌 수 없음
  ```

### 세미콜론과 세미콜론 자동 삽입 기능
+ 프리티어를 쓰자 


### 표현식인 문과 표현식이 아닌 문
+ 문은 표현식 이거나 아니거나로 나뉨
+ 표현식인지 알아보려면 변수에 할당 해보면됨
  ex)
   ```javascript
  var sum; //표현식이 아님 var foo = var sum; 에 할당이 안되기 때문에
  ```
+ 위와 같은 이유로 변수 할당문은 식이 아닌 문임.
